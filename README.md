# Math-IA-Spherical-Mirror
This repository contains contents of my Mathematics Internal Assessment focused on modeling reflections of lightrays from a spherical mirror through vector algebra.
Firstly, **the law of reflection** is derived. That is, the vector of a reflected ray can be expressed as a function of a vector of incoming lightray and a surface normal.
This function is subsequently used to explore how the origin of incoming ray and its direction vector are related to the ray reflected from an ideal concave hemispherical mirror with radius of 1.
